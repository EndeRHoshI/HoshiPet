# 🐱 HoshiPet v1.1.0 — 长线经济重塑 & UI 交互重构

> 🎮 **[在线游玩](https://EndeRHoshI.github.io/HoshiPet/)**

本次更新围绕「长线生存体验」和「操作交互品质」进行了大规模重构。

---

## 🆕 新增功能

### 🎯 首页日程系统重构
- **状态总览卡片**：新增「⏳ 当前在做什么」独立卡片，实时展示打工倒计时、学习进度等信息
- **弹窗式职业选择**：学习 / 打工不再内嵌页面底部，改为清爽的全屏 Modal 弹窗
- **智能防呆 Toast**：重复点击当前状态时，底部悬浮气泡提示，不污染日志
- 支持打工中直接换岗、学习中直接换课，无需先切回休息

### 🔐 开发者认证网关
- 调试面板（刷新速度、暂停、金币）默认隐藏
- 通过密码验证后解锁，解锁状态跨会话持久化

## ⚖️ 数值平衡调整

### 🕐 长线生存模型
- 代谢速度全面放缓至 1/50，实现「1天微降、2天中等、3天极限」的真实节奏
- 1 Tick 映射现实 30 秒，所有时间 UI 换算为人类可读的时/分/秒
- **移除 8 小时离线上限**，实现无尽挂机

### 💪 疲劳博弈机制
- 打工状态：健康额外 -0.045/tick，心情额外 -0.055/tick
- 学习状态：健康额外 -0.025/tick，心情额外 -0.035/tick
- 强制玩家在「赚钱」与「休息回血」之间策略博弈

### 🛒 商品体系重构
- 全部 24 种物品引入多维度负面词条（高热量降健康、洗澡降心情等）
- 猫咪保健品 → **猫咪疫苗**（心情 +0 → -20）
- 冻干益生菌健康 +60 → **+20**
- 宠物医院体检健康 +100 → **+80**
- 初始资金提升至 **100 金币**

## 🐛 Bug 修复 & 体验优化

- 修复背包使用物品时弹窗闪烁/重渲染问题（改为 DOM 原地刷新）
- 修复 Modal 遮罩层触摸穿透导致主界面误滑
- 修复重新进入页面时日志偶尔显示空白
- 商城分类默认折叠，节省屏幕空间
- 移除商城顶部冗余标题
- 职业/打工卡片强制双栏栅格布局，减少留白
