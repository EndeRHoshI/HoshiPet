# 🐱 HoshiPet v1.0.0 — 首个稳定版本

> 🎮 **[在线游玩](https://EndeRHoshI.github.io/HoshiPet/)**

HoshiPet (星宠) 首个正式发布版本，一款基于纯静态前端技术栈打造的网页端宠物养成与放置生存游戏。

---

## ✨ 核心功能

### 🐾 宠物系统
- 11 种猫咪品种（含 5 种稀有品种），每只猫拥有独立的性别、活泼度、抵抗力、智力属性
- 五维状态系统：饱食、水分、清洁、健康、心情，实时衰减与动态心情反馈
- 猫咪死亡与复活机制

### 🎒 物品与商城
- 24 种商城物品，覆盖食品、饮料、清洁、医疗、玩具、特殊道具六大类别
- 背包系统，支持使用物品并实时反馈效果
- 改名卡、复活卡等特殊道具

### 💼 职业系统
- 多阶段职业学习树，支持从基础到高级职业的晋级路线
- 打工赚金币机制，高阶职业收入更高
- 学习进度实时追踪

### 🎨 UI/UX 设计
- 日系毛玻璃卡片画风，4K 高清背景
- 完整的移动端响应式适配
- 半透明浮动三列布局

### 🔧 核心引擎
- 基于 Tick 的心跳算法引擎
- `Page Visibility API` 驱动的离线追帧补偿系统
- `localStorage` 全状态持久化存档

## 📦 技术栈

`HTML5` · `CSS3` · `Vanilla JavaScript` · 零构建依赖 · GitHub Pages 部署
