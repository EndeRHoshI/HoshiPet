# HoshiPet (星宠) 🐱✨

HoshiPet 是一款基于纯静态前端技术栈打造的**网页端宠物养成与放置生存游戏**。
游戏主打温暖治愈的"日系毛玻璃卡片"画风，拥有严密的物理时间换算系统、疲劳干预机制与长线物价经济学。你可以在电脑和手机浏览器上沉浸式体验饲养、打工、学习与收集的乐趣。

## 💡 核心实现原理

### 1. Tick Engine 与时间映射
底层心跳算法将程序的离散 Tick（默认 1 Tick = 30 秒）折算为人类可读的"时 / 分 / 秒"。配合 `Page Visibility API`，用户息屏或切到后台时，系统会在页面再度可见的瞬间执行**智能离线追帧算法**，一口气结算离线期间的全部状态衰减与收益，无时间上限。

### 2. 疲劳博弈经济学 (Fatigue Economics)
颠覆传统"无脑放置"。空闲待机时生理指征可安稳续航约三天；而打工/学习状态会触发额外的健康与心情衰减压力（打工 > 学习 > 空闲），迫使玩家在"赚钱"与"养生"之间持续策略博弈。

### 3. 动态制衡商城 (Dynamic Trade-offs)
所有物品都携带多维度正负面词条。高热量零食补饱腹却损健康和清洁；洗澡大幅提升清洁却闪崩心情；疫苗回血健康但心情暴跌。玩家必须根据五维短板组合配比"连招组合拳"以维持长线存活。

### 4. 状态机与日程调度
猫咪行为由有限状态机驱动（`idle` / `work` / `study` / `trip`），通过弹窗式 Modal 选择具体职业后切换状态。支持打工中直接换岗、学习中直接换课，无需先回到空闲态。

### 5. 物理级时停
全局单例暂停开关持久化至存档，暂停状态下完全免疫离线计算，实现真正的"冻结时间线"长期休眠。
