# HoshiPet 猫咪养成游戏
一款温暖有趣的网页版猫咪养成模拟器。🐱
基础机制：每次经过一个最小时间（一个 tick，调试阶段可以调整，最后找到一个比较合适的值），会让宠物发生一些变化，增减随机数值，完成打工、学习，发生随机事件

## 场景
### 场景一 领养宠物
选择宠物，生成六个宠物选项，有性别、品种（固定一些品种，其中放入稀有品种：曼基康、金银渐层等）、基础属性（影响五维的增长），并赋予命名

基础属性：活泼冷静、抵抗力强弱、聪明笨拙，会影响五维的增长速率

### 场景二 自己的小屋
状态变化除了在线的变化，还要支持离线状态变化，用户关闭页面或退出程序后，再次打开，要能够再次读取到数据，并且计算期间发生的变化

宠物基础数值有：饱食、口渴、清洁、健康、心情，下面都简称五维，多维度表示宠物状态，饱食、口渴、清洁随时间减少随机值，健康根据上面三个随时间减少随机值，心情根据上面四个随时间减少随机值

宠物有下面几种状态：
* 状态一，空闲状态，正常消耗
* 状态二，打工系统，可以选择一些行为去赚钱，消耗会加快
* 状态三，学习系统，可以进行学习，学习后能掌握一些技能，能够赚更多的钱，学习过程中消耗会加快
* 状态四，出游，饱食、口渴、清洁消耗加速，心情健康上升，且有随机事件，会获得随机物品或者疾病，五维低于一定的值后会回家

### 场景三 商城、物品系统：
1. 有对应的物品可以增加五维数据（生成一些物品如：肥皂、洗发水、牙膏牙刷套装）
2. 一些功能性物品，比如改名、复活
